주요 기능

차분하고 꾸준한 감정 기록을 위한 모든 것

💙

6가지 감정 기록

괜찮음, 슬픔, 분노, 지침, 불안, 기타. 가볍게 선택하고 한 줄로 남기세요.

🐠

어항 성장 시스템

연속 기록 스트릭에 따라 알 → 치어 → 풍성한 어항으로 성장합니다.

🔍

히스토리 검색

날짜, 태그, 감정별로 손쉽게 찾고, 다시 보고 싶은 기록은 핀 고정.

📊

회복 인사이트

이번 주 회복 리포트와 감정 변화 흐름을 차분하게 보여드립니다.

☁️

iCloud 자동 동기화

기기 간 자동 동기화 + 수동 백업으로 소중한 기록을 안전하게.

🔒

Face ID 잠금

앱 전체 또는 민감 화면만 생체 인증으로 보호합니다.

🌐

40개 언어 지원

한국어, 영어, 일본어를 포함한 40개 언어로 즉시 전환됩니다.

🔔

하루 한 번 리마인더

편한 시간에 부드럽게 알려주고, 문구 스타일도 선택할 수 있어요.

📱

위젯 & Siri 단축어

홈/잠금화면 위젯과 Siri로 더 빠르게 기록을 시작하세요.

보안 및 개인정보

당신의 기록은 오직 당신만의 것입니다

로컬 우선 저장

모든 기록은 기기 내 로컬에 안전하게 저장됩니다.

iCloud CloudKit

Apple iCloud를 통해서만 동기화되며, 외부 서버 전송 없음.

Face ID / Touch ID

생체 인증으로 앱 전체 또는 특정 화면을 잠글 수 있습니다.

수동 백업 지원

JSON 백업 파일로 직접 내보내고 복원할 수 있습니다.

Apple 로그인

최초 1회 Apple 로그인으로 안전하게 시작합니다.

개인 정보 비전송

개인 기록은 서버로 전송되지 않습니다.
